# TranslateX

TranslateX is a lightweight translation application that provides fast and accurate multilingual text translation using the free-tier Gemini-3.5 API.


- Translate text between multiple languages
- Fast and responsive interface
- AI-powered translations using Gemini API
- Simple and clean user experience
- REST API support with FastAPI backend
